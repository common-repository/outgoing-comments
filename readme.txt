== Description ==

Display your recent outgoing comments on your dashboard menu to help you revisit your friend's site. The main purpose is to trackback your own comments so you can know where sites you have comment on (with BackType service).

It will keep your conversation and help you make good relationship. You can also put it on sidebar with OutGoing Comment Widget in this packet.



== Installation ==


1. Upload the zipped file to yoursite/wp-content/plugins
1. Activate the plugin through the 'Plugins' menu in WordPress
